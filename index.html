Etapa de Ideación

La idea general consiste en describir la dinámica de un sistema físico mediante la relación entre una variable dependiente, que representa el fenómeno observado (por ejemplo, el desplazamiento vertical de un puente colgante), y una variable independiente, como el tiempo.

Pregunta Esencial

¿Cómo obtener el modelo matemático que describe la dinámica entre las dos variables?

Elementos Claves

  • Variable dependiente: Desplazamiento vertical (u) del puente
  • Variable independiente: Tiempo (t)
  • Dinámica: Oscilaciones gobernadas por las leyes del movimiento armónico

Parámetros del Puente Q'eswachaka

Parámetro Valor Unidad
Longitud del puente 28 m
Masa por metro 19.85 kg/m
Rigidez estructural 224,284 N/m
Coeficiente de amortiguamiento 1,000 Ns/m
Amplitud de fuerza externa 1,000 N
Frecuencia angular 2π rad/s

Ecuación Diferencial del Sistema

La dinámica del puente se modela mediante una ecuación diferencial lineal de segundo orden que representa un sistema oscilatorio forzado y amortiguado:

m d²u/dt² + c du/dt + k u = F₀ cos(ωt)

Donde:

  • m: masa del puente o de una sección representativa
  • c: coeficiente de amortiguamiento
  • k: constante de rigidez estructural
  • F₀ cos(ωt): fuerza externa aplicada sobre el puente

Interpretación Física

Esta ecuación describe cómo el sistema responde ante perturbaciones externas, permitiendo analizar fenómenos como:

  • Resonancia: Cuando la frecuencia de la fuerza externa coincide con la frecuencia natural del sistema
  • Amortiguamiento: Reducción gradual de las oscilaciones debido a la fricción
  • Estabilidad: Capacidad del sistema para regresar a su estado de equilibrio

Resolución de la Ecuación Diferencial

Paso 1: Solución Homogénea (Yₕ)

Resolvemos la ecuación homogénea (sin fuerza externa):

m d²Yₕ/dt² + c dYₕ/dt + k Yₕ = 0

Obtenemos raíces complejas: r = -25.19 ± 103.27i

Solución general homogénea:

Yₕ(t) = e^(-25.19t) [C₁ cos(103.27t) + C₂ sin(103.27t)]

Paso 2: Solución Particular (Yₚ)

Para la ecuación no homogénea (con fuerza externa):

m d²Yₚ/dt² + c dYₚ/dt + k Yₚ = F₀ cos(ωt)

Proponemos: Yₚ(t) = A cos(ωt) + B sin(ωt)

Obtenemos: A ≈ 0.00447 m, B ≈ 0.0001256

Solución particular:

Yₚ(t) = 0.00447 cos(2πt) + 0.0001256 sin(2πt)

Paso 3: Solución General

Combinamos ambas soluciones:

Y(t) = Yₕ(t) + Yₚ(t)

Aplicamos condiciones iniciales Y(0) = 0 y Y'(0) = 0

Solución final:

Y(t) = e^(-25.19t) [-0.00447 cos(103.27t) - 0.001098 sin(103.27t)] + 0.00447 cos(2πt) + 0.0001256 sin(2πt)

Gráfico de la Solución

Interpretación: La respuesta transitoria (exponencial decreciente) desaparece rápidamente, dejando únicamente la respuesta estacionaria (oscilaciones periódicas).

Simulador Interactivo

Se ha desarrollado un prototipo de simulación que implementa el modelo matemático de oscilaciones del Puente Q'eswachaka.

Características Principales

  • Simulación en Tiempo Real: Resolución numérica usando el método Runge-Kutta de 4to orden
  • Visualización Múltiple: Gráficos de desplazamiento, velocidad y fuerza vs tiempo
  • Parámetros Configurables: Masa, amortiguamiento, rigidez, fuerza externa

Validación del Modelo

El prototipo implementa la ecuación diferencial exacta planteada:

m d²Y/dt² + c dY/dt + k Y = F₀ cos(ωt)

El simulador constituye una evidencia tangible de la aplicación práctica del modelamiento matemático propuesto, cumpliendo con los objetivos educativos y técnicos del proyecto.

Conclusión

Este proyecto demostró con éxito la capacidad de modelar matemáticamente las oscilaciones de un puente colgante utilizando una ecuación diferencial de segundo orden que considera la masa, la rigidez, el amortiguamiento y las fuerzas externas periódicas.

El modelo, validado a través de una simulación interactiva desarrollada por el equipo, permite analizar el comportamiento del puente bajo diferentes condiciones, incluyendo variaciones en la fuerza del viento, la rigidez estructural y el nivel de amortiguamiento.

El proyecto destaca la utilidad de las ecuaciones diferenciales en la resolución de problemas de ingeniería y la importancia del modelado matemático para la predicción y prevención de situaciones de riesgo.
